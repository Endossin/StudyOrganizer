# 📚 StudyOrganizer

O **StudyOrganizer** é uma aplicação completa (Full Stack) desenvolvida para auxiliar na organização e gerenciamento de rotinas de estudos. O projeto conta com uma API robusta em C# e uma interface moderna responsiva em Angular.

---

## 🛠️ Tecnologias Utilizadas

### **Back-end**
* **Linguagem:** C#
* **Framework:** .NET Core / Web API
* **IDE:** Visual Studio

### **Front-end**
* **Framework:** Angular
* **Linguagem:** TypeScript / HTML / SCSS
* **Editor:** VS Code

---

## 📂 Estrutura do Repositório

* `StudyOrganizer.API/`: Contém toda a lógica de servidor, rotas e integração com banco de dados.
* `StudyOrganizer.Web/`: Contém a interface do usuário desenvolvida em Angular.

---
